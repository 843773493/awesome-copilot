# GitHub Copilot 配方手册

一个用于使用 GitHub Copilot 工具和功能的实用配方和示例集合。每个配方提供专注于具体任务、可直接复制粘贴的代码片段和现实世界的示例，以帮助您完成常见操作。

## 配方手册内容

配方手册按工具或产品分类整理，适用语言的配方会单独收集：

### GitHub Copilot SDK

适用于多种语言的 GitHub Copilot SDK 开发配方。

- **[Copilot SDK 配方手册](copilot-sdk/)** - 包含 .NET、Go、Node.js 和 Python 的配方
  - 错误处理、会话管理、文件操作等
  - 每种语言的可运行示例
  - 最佳实践和完整实现指南

## 快速入门

1. 浏览与您要构建内容匹配的工具或产品文件夹
2. 查找解决您使用场景的配方
3. 复制代码片段，或查看 `recipe/` 子文件夹中的完整、可运行示例
4. 参考特定语言的文档，获取设置和执行说明

## 计划扩展

配方手册旨在与 GitHub Copilot 生态系统同步扩展。未来可能增加的配方包括：

- 其他 Copilot 工具和集成
- 高级模式和工作流程
- 与外部服务和 API 的集成
- 语言特定的优化和最佳实践
