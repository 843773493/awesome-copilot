# GitHub Copilot SDK 实用指南 — Go

此文件夹包含使用 Go 语言与 GitHub Copilot SDK 交互的简短实用示例。每个示例都简洁明了，可直接复制粘贴，并指向更完整的示例和测试。

## 示例

- [错误处理](error-handling.md): 优雅处理错误，包括连接失败、超时和清理操作。
- [多会话管理](multiple-sessions.md): 同时管理多个独立对话。
- [本地文件管理](managing-local-files.md): 利用 AI 驱动的分组策略按元数据组织文件。
- [PR 可视化](pr-visualization.md): 使用 GitHub MCP 服务器生成交互式的 PR 年龄图表。
- [会话持久化](persisting-sessions.md): 在重启之间保存并恢复会话。
