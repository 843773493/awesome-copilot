# GitHub Copilot SDK 食谱 — Python

此文件夹包含使用 GitHub Copilot SDK 与 Python 进行交互的简短、实用的示例。每个示例都简洁明了，可直接复制粘贴，并指向更完整的示例和测试。

## 示例

- [错误处理](error-handling.md): 优雅地处理错误，包括连接失败、超时和清理操作。
- [多个会话](multiple-sessions.md): 同时管理多个独立对话。
- [管理本地文件](managing-local-files.md): 使用 AI 驱动的分组策略按元数据组织文件。
- [PR 可视化](pr-visualization.md): 使用 GitHub MCP Server 生成交互式 PR 年龄图表。
- [会话持久化](persisting-sessions.md): 在重启之间保存并恢复会话。
