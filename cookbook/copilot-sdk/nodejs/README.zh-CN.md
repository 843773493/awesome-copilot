# GitHub Copilot SDK 实践手册 — Node.js / TypeScript

此文件夹托管用于与 Node.js/TypeScript 一起使用 GitHub Copilot SDK 的简短实用食谱。每个食谱都简洁明了，可直接复制粘贴，并指向更完整的示例和测试。

## 食谱

- [错误处理](error-handling.md): 优雅地处理错误，包括连接失败、超时和清理操作。
- [多会话管理](multiple-sessions.md): 同时管理多个独立的对话会话。
- [管理本地文件](managing-local-files.md): 使用 AI 驱动的分组策略按元数据组织文件。
- [Pull Request 可视化](pr-visualization.md): 使用 GitHub MCP Server 生成交互式的 Pull Request 年龄图表。
- [会话持久化](persisting-sessions.md): 在重启之间保存并恢复会话。
