# GitHub Copilot SDK 烹饪指南 — .NET（C#）

此文件夹包含使用 GitHub Copilot SDK 与 .NET 进行交互的简短且实用的食谱。每个食谱都简洁明了，可直接复制粘贴使用，并指向更完整的示例和测试。

## 食谱

- [错误处理](error-handling.md)：优雅地处理错误，包括连接失败、超时和清理操作。
- [管理多个会话](multiple-sessions.md)：同时管理多个独立的对话会话。
- [管理本地文件](managing-local-files.md)：通过人工智能驱动的分组策略，按元数据组织文件。
- [Pull Request 可视化](pr-visualization.md)：使用 GitHub MCP 服务器生成交互式的 Pull Request 年龄图表。
- [会话持久化](persisting-sessions.md)：在重启之间保存并恢复会话。

## 贡献

通过在此文件夹中创建一个 markdown 文件并将其链接到上方来添加新的食谱。请遵循 [CONTRIBUTING.md](../../../CONTRIBUTING.md) 中的仓库指南。

## 状态

这些食谱现已完成并可直接使用；烹饪指南将继续随着新场景的添加而演进。
