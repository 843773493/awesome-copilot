# Power Apps 代码应用开发插件

专为 Power Apps 代码应用开发打造的完整工具包，包含项目脚手架搭建、开发规范以及专家指导，帮助您构建与 Power Platform 集成的代码优先应用。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install power-apps-code-apps@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/power-apps-code-apps:power-apps-code-app-scaffold` | 使用 PAC CLI 配置、SDK 集成和连接器设置，生成完整的 Power Apps 代码应用项目 |

### 代理

| 代理 | 描述 |
|------|------|
| `power-platform-expert` | 提供关于代码应用、画布应用、Dataverse、连接器以及 Power Platform 最佳实践的指导的专家 |

## 来源

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个由社区驱动的 GitHub Copilot 扩展集合。
