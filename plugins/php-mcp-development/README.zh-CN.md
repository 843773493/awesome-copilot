# PHP MCP 服务器开发插件

使用官方 PHP SDK 进行基于属性的发现，构建模型上下文协议服务器的全面资源，包括最佳实践、项目生成和专家协助

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install php-mcp-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/php-mcp-development:php-mcp-server-generator` | 使用官方 PHP SDK 生成一个完整的 PHP 模型上下文协议服务器项目，包含工具、资源、提示和测试 |

### 代理

| 代理 | 描述 |
|------|------|
| `php-mcp-expert` | 使用官方 PHP SDK 进行基于属性发现的 PHP MCP 服务器开发专家助手 |

## 来源

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，一个由社区驱动的 GitHub Copilot 扩展集合。
