# Go MCP 服务器开发插件

使用官方 github.com/modelcontextprotocol/go-sdk 构建模型上下文协议（MCP）服务器的完整工具包。包含最佳实践指南、生成服务器的提示命令以及专家聊天模式以提供指导。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install go-mcp-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/go-mcp-development:go-mcp-server-generator` | 使用官方 github.com/modelcontextprotocol/go-sdk 生成一个完整的 Go MCP 服务器项目，包含正确的结构、依赖项和实现。 |

### 代理

| 代理 | 描述 |
|------|------|
| `go-mcp-expert` | 使用官方 SDK 构建模型上下文协议（MCP）服务器的专家助手。 |

## 来源

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个由社区驱动的 GitHub Copilot 扩展集合。
