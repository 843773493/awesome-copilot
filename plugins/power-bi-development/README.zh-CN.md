# Power BI 开发插件

涵盖数据建模、DAX 优化、性能调优、可视化设计、安全最佳实践以及 DevOps/ALM 指南的全面 Power BI 开发资源，用于构建企业级 Power BI 解决方案。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install power-bi-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/power-bi-development:power-bi-dax-optimization` | 全面的 Power BI DAX 公式优化提示，用于提高 DAX 计算的性能、可读性和可维护性。 |
| `/power-bi-development:power-bi-model-design-review` | 全面的 Power BI 数据模型设计评审提示，用于评估模型架构、关系及优化机会。 |
| `/power-bi-development:power-bi-performance-troubleshooting` | 系统化的 Power BI 性能故障排查提示，用于识别、诊断和解决 Power BI 模型、报告和查询中的性能问题。 |
| `/power-bi-development:power-bi-report-design-consultation` | Power BI 报告可视化设计提示，用于创建高效、用户友好且易于访问的报告，优化图表选择和布局设计。 |

### 代理（Agent）

| 代理 | 描述 |
|------|------|
| `power-bi-data-modeling-expert` | 使用星型架构原则、关系设计和微软最佳实践，提供 Power BI 数据建模专家指导，以实现最佳模型性能和可用性。 |
| `power-bi-dax-expert` | 使用微软最佳实践，提供 Power BI DAX 专家指导，以提升 DAX 公式和计算的性能、可读性和可维护性。 |
| `power-bi-performance-expert` | 提供 Power BI 性能优化专家指导，用于排查、监控和提升 Power BI 模型、报告和查询的性能。 |
| `power-bi-visualization-expert` | 使用微软最佳实践，提供 Power BI 报告设计和可视化专家指导，创建高效、性能优越且用户友好的报告和仪表板。 |
