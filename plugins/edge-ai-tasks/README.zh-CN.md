# microsoft/edge-ai 插件的任务

面向中级到高级用户的任务研究员和任务规划器，适用于大型代码库 - 由 microsoft/edge-ai 提供

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install edge-ai-tasks@awesome-copilot
```

## 包含内容

### 代理

| 代理 | 描述 |
|-------|-------------|
| `task-researcher` | 专为全面项目分析设计的任务研究员专家 - 由 microsoft/edge-ai 提供 |
| `task-planner` | 用于创建可执行实施计划的任务规划器 - 由 microsoft/edge-ai 提供 |

## 来源

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个由社区驱动的 GitHub Copilot 扩展集合。
