# DevOps 值班插件

一组专注于事件排查的提示、指令和聊天模式，帮助快速使用 DevOps 工具和 Azure 资源响应问题。

## 安装

```bash
# 使用 Copilot CLI 安装
copilot plugin install devops-oncall@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/devops-oncall:azure-resource-health-diagnose` | 分析 Azure 资源健康状况，从日志和遥测数据中诊断问题，并为识别出的问题创建修复计划。 |
| `/devops-oncall:multi-stage-dockerfile` | 为任何语言或框架创建优化的多阶段 Dockerfile |

### 代理

| 代理 | 描述 |
|------|------|
| `azure-principal-architect` | 使用 Azure 架构最佳实践框架和微软最佳实践原则，提供 Azure 主体架构专家指导。 |

## 来源

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个社区驱动的 GitHub Copilot 扩展集合。
