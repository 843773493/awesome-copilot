# 从OpenAPI生成应用 - Node.js NestJS插件

从OpenAPI规范生成生产就绪的NestJS应用。包含项目脚手架、控制器和服务生成、TypeScript最佳实践以及企业级模式。

## 安装

```bash
# 使用Copilot CLI
copilot plugin install openapi-to-application-nodejs-nestjs@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/openapi-to-application-nodejs-nestjs:openapi-to-application-code` | 从OpenAPI规范生成一个完整的、可投入生产的应用 |

### 代理

| 代理 | 描述 |
|------|------|
| `openapi-to-application` | 根据OpenAPI规范生成可运行应用的专家助手 |
