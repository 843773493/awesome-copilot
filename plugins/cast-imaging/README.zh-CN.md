# CAST Imaging 代理插件

使用 CAST Imaging 进行软件分析、变更影响评估、结构质量建议和架构审查的专用代理综合集合。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install cast-imaging@awesome-copilot
```

## 包含内容

### 代理

| 代理 | 描述 |
|------|------|
| `cast-imaging-software-discovery` | 通过使用 CAST Imaging 进行静态代码分析，实现对软件应用程序的全面发现和架构映射的专用代理 |
| `cast-imaging-impact-analysis` | 通过使用 CAST Imaging 对软件系统进行全面变更影响评估和风险分析的专用代理 |
| `cast-imaging-structural-quality-advisor` | 通过使用 CAST Imaging 识别、分析并提供代码质量问题修复指导的专用代理 |

## 来源

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个由社区驱动的 GitHub Copilot 扩展集合。
