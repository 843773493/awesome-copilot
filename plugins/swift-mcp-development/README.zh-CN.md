# Swift MCP服务器开发插件

使用官方MCP Swift SDK包，结合现代并发特性，构建Model Context Protocol服务器的全面集合。

## 安装

```bash
# 使用Copilot CLI
copilot plugin install swift-mcp-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/swift-mcp-development:swift-mcp-server-generator` | 使用官方MCP Swift SDK包生成完整的Model Context Protocol服务器项目。 |

### 代理

| 代理 | 描述 |
|------|------|
| `swift-mcp-expert` | 使用现代并发特性和官方MCP Swift SDK，为构建Model Context Protocol服务器提供专家级帮助的Swift MCP专家代理。 |

## 源码

此插件是[Awesome Copilot](https://github.com/github/awesome-copilot)的一部分，这是一个由社区驱动的GitHub Copilot扩展集合。
