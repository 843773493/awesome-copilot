# OpenAPI到应用 - Java Spring Boot插件

从开放API规范生成生产就绪的Spring Boot应用。包含项目脚手架生成、REST控制器生成、服务层组织以及Spring Boot最佳实践。

## 安装

```bash
# 使用Copilot CLI
copilot plugin install openapi-to-application-java-spring-boot@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/openapi-to-application-java-spring-boot:openapi-to-application-code` | 从开放API规范生成一个完整且生产就绪的应用程序 |

### 代理

| 代理 | 描述 |
|------|------|
| `openapi-to-application` | 用于根据开放API规范生成可运行应用程序的专家助手 |
