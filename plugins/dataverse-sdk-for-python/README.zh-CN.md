# Dataverse SDK Python插件

构建生产就绪的Python与Microsoft Dataverse集成的全面工具集。包含官方文档、最佳实践、高级功能、文件操作以及代码生成提示。

## 安装

```bash
# 使用Copilot CLI
copilot plugin install dataverse-sdk-for-python@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/dataverse-sdk-for-python:dataverse-python-quickstart` | 使用官方模式生成Python SDK的设置 + CRUD + 批处理 + 分页代码片段。 |
| `/dataverse-sdk-for-python:dataverse-python-advanced-patterns` | 使用高级模式、错误处理和优化技术生成Dataverse SDK的生产代码。 |
| `/dataverse-sdk-for-python:dataverse-python-production-code` | 使用Dataverse SDK生成包含错误处理、优化和最佳实践的生产就绪Python代码 |
| `/dataverse-sdk-for-python:dataverse-python-usecase-builder` | 为特定的Dataverse SDK用例生成完整解决方案及架构建议 |
