# Java MCP 服务器开发插件

使用官方 MCP Java SDK 通过响应式流和 Spring Boot 集成构建 Model Context Protocol 服务器的完整工具包。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install java-mcp-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/java-mcp-development:java-mcp-server-generator` | 使用官方 MCP Java SDK 通过响应式流生成一个完整的 Model Context Protocol 服务器项目，并可选集成 Spring Boot。 |

### 代理

| 代理 | 描述 |
|------|------|
| `java-mcp-expert` | 使用响应式流、官方 MCP Java SDK 和 Spring Boot 集成构建 Model Context Protocol 服务器的专家协助。 |

## 源代码

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个由社区驱动的 GitHub Copilot 扩展集合。
