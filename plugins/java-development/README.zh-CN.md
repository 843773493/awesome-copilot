# Java开发插件

全面的提示和指令集合，涵盖Spring Boot、Quarkus、测试、文档以及最佳实践等Java开发相关内容。

## 安装

```bash
# 使用Copilot CLI
copilot plugin install java-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/java-development:java-docs` | 确保Java类型使用Javadoc注释进行文档记录，并遵循文档的最佳实践。 |
| `/java-development:java-junit` | 获取JUnit 5单元测试的最佳实践，包括数据驱动测试。 |
| `/java-development:java-springboot` | 获取使用Spring Boot开发应用程序的最佳实践。 |
| `/java-development:create-spring-boot-java-project` | 创建Spring Boot Java项目骨架 |
