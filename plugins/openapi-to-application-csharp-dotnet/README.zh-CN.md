# 从OpenAPI生成应用程序 - C# .NET插件

根据OpenAPI规范生成生产就绪的.NET应用程序。包含ASP.NET Core项目骨架生成、控制器生成、Entity Framework集成以及C#最佳实践。

## 安装

```bash
# 使用Copilot CLI
copilot plugin install openapi-to-application-csharp-dotnet@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/openapi-to-application-csharp-dotnet:openapi-to-application-code` | 根据OpenAPI规范生成一个完整且可直接投入生产的应用程序 |

### 代理

| 代理 | 描述 |
|------|------|
| `openapi-to-application` | 根据OpenAPI规范生成可运行应用程序的专家代理 |
