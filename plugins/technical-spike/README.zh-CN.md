# 技术尖峰插件

用于创建、管理和研究技术尖峰的工具，以减少在进行解决方案的规格说明和实现之前未知数和假设。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install technical-spike@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/technical-spike:create-technical-spike` | 创建时间限制的技术尖峰文档，用于在实施前研究和解决关键开发决策。 |

### 代理

| 代理 | 描述 |
|------|------|
| `research-technical-spike` | 通过彻底的调查和受控实验，系统地研究和验证技术尖峰文档。 |

## 源码

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个社区驱动的 GitHub Copilot 扩展集合。
