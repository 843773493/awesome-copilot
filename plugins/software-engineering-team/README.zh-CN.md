# 软件工程团队插件

涵盖从用户体验设计和架构到安全性和 DevOps 的完整软件开发生命周期的 7 个专业代理。

## 安装

```bash
# 使用 Copilot CLI 安装
copilot plugin install software-engineering-team@awesome-copilot
```

## 包含内容

### 代理

| 代理 | 描述 |
|------|------|
| `se-ux-ui-designer` | 用于 Figma 和设计流程的 Jobs-to-be-Done 分析、用户旅程映射和用户体验研究工件 |
| `se-technical-writer` | 技术写作专家，负责创建开发者文档、技术博客、教程和教育内容 |
| `se-gitops-ci-specialist` | DevOps 专家，专注于 CI/CD 管道、部署调试和 GitOps 工作流，使部署变得乏味且可靠 |
| `se-product-manager-advisor` | 产品管理顾问，提供创建 GitHub 问题、将业务价值与用户需求对齐以及制定数据驱动产品决策的指导 |
| `se-responsible-ai-code` | 负责任的 AI 专家，通过防止偏见、符合可访问性标准、伦理开发和包容性设计，确保 AI 为所有人服务 |
| `se-system-architecture-reviewer` | 系统架构审查专家，具备 Well-Architected 框架、设计验证和针对 AI 与分布式系统的可扩展性分析 |
| `se-security-reviewer` | 专注于安全的代码审查专家，涵盖 OWASP Top 10、零信任、LLM 安全性和企业安全标准 |

## 来源

该插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个由社区驱动的 GitHub Copilot 扩展集合。
