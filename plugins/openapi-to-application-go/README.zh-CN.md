# 从 OpenAPI 生成应用 - Go 插件

从 OpenAPI 规范生成生产就绪的 Go 应用程序。包含项目骨架搭建、处理程序生成、中间件配置以及 REST API 的 Go 最佳实践。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install openapi-to-application-go@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/openapi-to-application-go:openapi-to-application-code` | 从 OpenAPI 规范生成完整且可投入生产的应用程序 |

### 代理

| 代理 | 描述 |
|------|------|
| `openapi-to-application` | 从 OpenAPI 规范生成可运行应用程序的专家助手 |

## 来源

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个社区驱动的 GitHub Copilot 扩展集合。
