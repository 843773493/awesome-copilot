# 前端网页开发插件

现代前端网页开发所需的必备提示、指令和聊天模式，包括 React、Angular、Vue、TypeScript 和 CSS 框架。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install frontend-web-dev@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/frontend-web-dev:playwright-explore-website` | 使用 Playwright MCP 进行网站探索测试 |
| `/frontend-web-dev:playwright-generate-test` | 基于场景生成 Playwright 测试 |

### 代理

| 代理 | 描述 |
|------|------|
| `expert-react-frontend-engineer` | 专精于现代 hooks、Server Components、Actions、TypeScript 以及性能优化的 React 19.2 前端专家工程师 |
| `electron-angular-native` | 针对 Electron 应用的代码审查模式（主进程为 Node.js 后端，渲染进程为 Angular 前端，以及原生集成层（例如 AppleScript、shell 或原生工具链））。其他仓库中的服务不会在此处进行审查。 |

## 来源

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个由社区驱动的 GitHub Copilot 扩展集合。
