# Kotlin MCP服务器开发插件

使用官方 io.modelcontextprotocol:kotlin-sdk 库，构建 Model Context Protocol (MCP) 服务器的完整工具包。包含最佳实践指南、服务器生成提示以及专家聊天模式以提供指导。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install kotlin-mcp-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/kotlin-mcp-development:kotlin-mcp-server-generator` | 使用官方 io.modelcontextprotocol:kotlin-sdk 库生成一个完整的 Kotlin MCP 服务器项目，包含正确的结构、依赖项和实现。 |

### 代理

| 代理 | 描述 |
|------|------|
| `kotlin-mcp-expert` | 使用官方 SDK 构建 Model Context Protocol (MCP) 服务器的专家助手。 |

## 来源

本插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个由社区驱动的 GitHub Copilot 扩展集合。
