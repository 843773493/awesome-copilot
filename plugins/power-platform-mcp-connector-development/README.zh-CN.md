# Power Platform MCP连接器开发插件

用于在Microsoft Copilot Studio中开发带有模型上下文协议（Model Context Protocol）集成的Power Platform自定义连接器的完整工具包

## 安装

```bash
# 使用Copilot CLI
copilot plugin install power-platform-mcp-connector-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/power-platform-mcp-connector-development:power-platform-mcp-connector-suite` | 生成适用于Copilot Studio的完整Power Platform自定义连接器，包含模型上下文协议集成 - 包括模式生成、故障排除和验证 |
| `/power-platform-mcp-connector-development:mcp-copilot-studio-server-generator` | 生成一个针对Copilot Studio集成优化的完整MCP服务器实现，具备正确的模式约束和可流式传输的HTTP支持 |

### 代理

| 代理 | 描述 |
|------|------|
| `power-platform-mcp-integration-expert` | 在Copilot Studio中开发带有模型上下文协议集成的Power Platform自定义连接器方面的专家 - 对模式、协议和集成模式有全面的理解 |
