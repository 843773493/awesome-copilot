# 基于MCP的M365代理插件

用于构建基于模型上下文协议的Microsoft 365 Copilot声明式代理的全面集合

## 安装

```bash
# 使用Copilot CLI
copilot plugin install mcp-m365-copilot@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/mcp-m365-copilot:mcp-create-declarative-agent` | 创建声明式代理 |
| `/mcp-m365-copilot:mcp-create-adaptive-cards` | 创建自适应卡片 |
| `/mcp-m365-copilot:mcp-deploy-manage-agents` | 部署和管理代理 |

### 代理

| 代理 | 描述 |
|------|------|
| `mcp-m365-agent-expert` | 用于构建基于模型上下文协议的Microsoft 365 Copilot声明式代理的专家助手 |

## 来源
