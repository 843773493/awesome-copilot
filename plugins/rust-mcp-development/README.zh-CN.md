# Rust MCP 服务器开发插件

使用官方 rmcp SDK 通过 async/await、过程宏 和类型安全实现，在 Rust 中构建高性能的 Model Context Protocol 服务器。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install rust-mcp-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/rust-mcp-development:rust-mcp-server-generator` | 使用官方 rmcp SDK 生成一个完整的 Rust Model Context Protocol 服务器项目，包含工具、提示、资源和测试 |

### 代理

| 代理 | 描述 |
|------|------|
| `rust-mcp-expert` | 使用 tokio 异步运行时的 Rust MCP 服务器开发专家助手 |

## 源码

该插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个社区驱动的 GitHub Copilot 扩展集合。
