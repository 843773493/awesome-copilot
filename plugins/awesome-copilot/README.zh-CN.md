# Awesome Copilot 插件

元提示语，帮助您发现并生成精选的 GitHub Copilot 代理、集合、指令、提示和技能。

## 安装

```bash
# 使用 Copilot CLI 安装
copilot plugin install awesome-copilot@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/awesome-copilot:suggest-awesome-github-copilot-collections` | 根据当前仓库上下文和聊天历史，从 awesome-copilot 仓库中推荐相关的 GitHub Copilot 集合，提供自动下载和安装集合资源，并识别需要更新的过时集合资源。 |
| `/awesome-copilot:suggest-awesome-github-copilot-instructions` | 根据当前仓库上下文和聊天历史，从 awesome-copilot 仓库中推荐相关的 GitHub Copilot 指令文件，避免与本仓库中已有的指令文件重复，并识别需要更新的过时指令。 |
| `/awesome-copilot:suggest-awesome-github-copilot-prompts` | 根据当前仓库上下文和聊天历史，从 awesome-copilot 仓库中推荐相关的 GitHub Copilot 提示文件，避免与本仓库中已有的提示文件重复，并识别需要更新的过时提示。 |
| `/awesome-copilot:suggest-awesome-github-copilot-agents` | 根据当前仓库上下文和聊天历史，从 awesome-copilot 仓库中推荐相关的 GitHub Copilot 自定义代理文件，避免与本仓库中已有的自定义代理重复，并识别需要更新的过时代理。 |

### 代理

| 代理 | 描述 |
|------|------|
| `meta-agentic-project-scaffold` | 元代理项目创建助手，帮助用户有效创建和管理项目工作流。 |

## 来源

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个社区驱动的 GitHub Copilot 扩展集合。
