# C# MCP服务器开发插件

使用官方SDK构建模型上下文协议（MCP）服务器的完整工具包。包含最佳实践指南、生成服务器的提示以及专家聊天模式以提供指导。

## 安装

```bash
# 使用Copilot CLI
copilot plugin install csharp-mcp-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/csharp-mcp-development:csharp-mcp-server-generator` | 使用工具、提示和正确配置生成完整的MCP服务器项目 |

### 代理

| 代理 | 描述 |
|------|------|
| `csharp-mcp-expert` | 用于开发模型上下文协议（MCP）服务器的专家助手 |

## 源码

该插件是[Awesome Copilot](https://github.com/github/awesome-copilot)的一部分，这是一个由社区驱动的GitHub Copilot扩展集合。
