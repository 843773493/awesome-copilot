# 从OpenAPI到应用 - Python FastAPI插件

从OpenAPI规范生成生产就绪的FastAPI应用。包含项目脚手架、路由生成、依赖注入以及异步API的Python最佳实践。

## 安装

```bash
# 使用Copilot CLI
copilot plugin install openapi-to-application-python-fastapi@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/openapi-to-application-python-fastapi:openapi-to-application-code` | 从OpenAPI规范生成一个完整且生产就绪的应用程序 |

### 代理

| 代理 | 描述 |
|------|------|
| `openapi-to-application` | 从OpenAPI规范生成可运行应用程序的专家助手 |
