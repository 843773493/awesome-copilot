# Ruby MCP服务器开发插件

使用官方MCP Ruby SDK gem构建Model Context Protocol服务器的完整工具包，并支持Rails集成。

## 安装

```bash
# 使用Copilot CLI
copilot plugin install ruby-mcp-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/ruby-mcp-development:ruby-mcp-server-generator` | 使用官方MCP Ruby SDK gem生成一个完整的Ruby Model Context Protocol服务器项目。 |

### 代理

| 代理 | 描述 |
|------|------|
| `ruby-mcp-expert` | 使用官方MCP Ruby SDK gem构建Ruby Model Context Protocol服务器的专家协助，并支持Rails集成。 |

## 来源

此插件是[Awesome Copilot](https://github.com/github/awesome-copilot)的一部分，这是一个由社区驱动的GitHub Copilot扩展集合。
