# 您需要哪些上下文信息？

在回答我的问题之前，请告诉我您需要查看哪些文件。

## 我的问题

{{question}}

## 操作说明

1. 根据我的问题，列出您需要检查的文件  
2. 说明每个文件的相关性  
3. 请注意在此对话中您已查看的文件  
4. 指出您不确定的内容

## 输出格式

```markdown
## 我需要的文件

### 必须查看的文件（准确回答所需）
- `path/to/file.ts` — [为何需要]

### 建议查看的文件（完整回答有帮助）
- `path/to/file.ts` — [为何有帮助]

### 已查看的文件
- `path/to/file.ts` — [来自对话早期]

### 不确定的内容
- [在未查看代码的情况下不确定的内容]
```

在我提供这些文件后，我会再次提出我的问题。
