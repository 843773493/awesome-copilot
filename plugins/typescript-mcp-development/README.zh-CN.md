# TypeScript MCP服务器开发插件

使用官方SDK在TypeScript/Node.js中构建模型上下文协议（MCP）服务器的完整工具包。包含最佳实践指南、生成服务器的提示以及专家聊天模式以提供指导。

## 安装

```bash
# 使用Copilot CLI安装
copilot plugin install typescript-mcp-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/typescript-mcp-development:typescript-mcp-server-generator` | 生成一个完整的TypeScript MCP服务器项目，包含工具、资源和正确配置 |

### 代理

| 代理 | 描述 |
|------|------|
| `typescript-mcp-expert` | 开发模型上下文协议（MCP）服务器的TypeScript专家助手 |
