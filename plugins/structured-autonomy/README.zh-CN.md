# 结构化自主插件

高级规划，高效实施

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install github/awesome-copilot/plugins/structured-autonomy
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/structured-autonomy:structured-autonomy-plan` | 结构化自主计划 |
| `/structured-autonomy:structured-autonomy-generate` | 结构化自主生成 |
| `/structured-autonomy:structured-autonomy-implement` | 结构化自主实施 |

## 源码

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个社区驱动的 GitHub Copilot 扩展集合。
