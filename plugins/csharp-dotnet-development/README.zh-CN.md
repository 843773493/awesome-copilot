# C# .NET 开发插件

C# 和 .NET 开发所需的提示、指令和聊天模式，涵盖测试、文档编写及最佳实践等内容。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install csharp-dotnet-development@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/csharp-dotnet-development:csharp-async` | 获取 C# 异步编程的最佳实践 |
| `/csharp-dotnet-development:aspnet-minimal-api-openapi` | 使用正确的 OpenAPI 文档创建 ASP.NET 最小 API 端点 |
| `/csharp-dotnet-development:csharp-xunit` | 获取 XUnit 单元测试的最佳实践，包括数据驱动测试 |
| `/csharp-dotnet-development:dotnet-best-practices` | 确保 .NET/C# 代码符合解决方案/项目的最佳实践 |
| `/csharp-dotnet-development:dotnet-upgrade` | 用于全面 .NET 框架升级分析与执行的即用型提示 |

### 代理

| 代理 | 描述 |
|------|------|
| `expert-dotnet-software-engineer` | 使用现代软件设计模式提供专业的 .NET 软件工程指导 |
