# 安全与代码质量插件

构建安全、可维护且高性能应用程序的安全框架、可访问性指南、性能优化及代码质量最佳实践。

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install security-best-practices@awesome-copilot
```

## 包含内容

### 命令（斜杠命令）

| 命令 | 描述 |
|------|------|
| `/security-best-practices:ai-prompt-engineering-safety-review` | 全面的AI提示工程安全审查与改进建议。分析提示的安全性、偏见、安全漏洞和有效性，同时提供详细的改进建议，涵盖广泛的框架、测试方法和教育内容。 |

## 源码

此插件是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，这是一个由社区驱动的 GitHub Copilot 扩展集合。
