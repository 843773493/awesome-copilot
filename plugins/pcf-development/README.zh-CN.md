# Power Apps 组件框架（PCF）开发插件

用于基于模型的应用和画布应用的 Power Apps 组件框架（PCF）自定义代码组件开发的完整工具包

## 安装

```bash
# 使用 Copilot CLI
copilot plugin install pcf-development@awesome-copilot
```
