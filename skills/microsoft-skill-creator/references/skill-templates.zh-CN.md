## 模板选择

| 技术类型 | 模板 | 示例 |
|----------|------|------|
| 客户端库、NuGet/npm 包 | SDK/库 | 语义内核、Azure SDK、MSAL |
| Azure 资源 | Azure 服务 | Cosmos DB、Azure Functions、应用服务 |
| 应用开发框架 | 框架/平台 | ASP.NET Core、Blazor、MAUI |
| REST API、协议、规范 | API/协议 | Microsoft Graph、OOXML、FHIR |

## 自定义指南

模板是起点。通过以下方式自定义：

1. **添加**针对技术独特方面的章节
2. **删除**不适用的章节
3. **根据复杂度调整深度**（对复杂技术添加更多概念）
4. **添加参考文件**用于无法放入 SKILL.md 的详细内容
5. **添加 sample_codes/** 用于超出内联片段的可运行示例
