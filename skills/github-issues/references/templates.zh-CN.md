

# 问题模板

复制并自定义这些模板用于问题正文。

## 报告错误模板

```markdown
## 描述
[对问题的清晰描述]

## 复现步骤
1. [第一步]
2. [第二步]
3. [依此类推...]

## 预期行为
[应该发生什么]

## 实际行为
[实际发生了什么]

## 环境
- 浏览器: [例如：Chrome 120]
- 操作系统: [例如：macOS 14.0]
- 版本: [例如：v1.2.3]

## 截图/日志
[如适用]

## 其他上下文
[任何其他相关信息]
```

## 功能请求模板

```markdown
## 摘要
[一行描述该功能]

## 动机
[为什么需要这个功能？它解决了什么问题？]

## 建议的解决方案
[该功能应该如何运作？]

## 验收标准
- [ ] [标准1]
- [ ] [标准2]
- [ ] [标准3]

## 已考虑的替代方案
[其他考虑过的方案及未选择的原因]

## 其他上下文
[原型图、示例或相关问题]
```

## 任务模板

```markdown
## 目标
[需要完成的任务]

## 详细信息
[工作的详细描述]

## 检查清单
- [ ] [子任务1]
- [ ] [子任务2]
- [ ] [子任务3]

## 依赖项
[任何阻塞项或相关工作]

## 备注
[其他上下文或注意事项]
```

## 最简模板

用于简单问题：

```markdown
## 描述
[什么和为什么]

## 任务
- [ ] [任务1]
- [ ] [任务2]
```
