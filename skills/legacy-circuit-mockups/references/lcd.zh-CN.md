# DFRobot FIT0127 字符液晶显示屏规格书

## 1. 概述

**DFRobot FIT0127** 是一种常见的 **兼容HD44780字符液晶模块**，广泛应用于嵌入式系统和爱好者项目中。这些显示屏使用点阵字符发生器实现字母数字输出，并支持8位和4位并行接口。

FIT0127模块通常与微控制器和8位CPU（如 **6502**、**AVR**、**PIC** 和 **Arduino** 平台）配合使用。

---

## 2. 一般特性

| 特性           | 描述                                         |
|----------------|-----------------------------------------------|
| 显示类型       | 字符液晶（STN，典型黄绿色背光）                |
| 控制器         | HD44780或兼容型号                              |
| 接口           | 并行（4位或8位）                               |
| 字符尺寸       | 5 x 8点阵                                     |
| 工作电压       | 5V逻辑（部分型号支持3.3V）                      |
| 背光           | LED（独立电源引脚）                            |
| 显示模式       | 双向透反射模式                                 |

---

## 3. 显示型号

FIT0127通常与 **16x2字符液晶模块** 相关联。

| 型号 | 字符数          |
|------|-----------------|
| FIT0127 | 16列 x 2行 |

## 4. 引脚配置

### 4.1 引脚分配（标准16针接口）

| 引脚 | 名称 | 描述           |
|-----:|------|----------------|
| 1    | VSS  | 地             |
| 2    | VDD  | +5V电源        |
| 3    | VO   | 对比度调节     |
| 4    | RS   | 寄存器选择     |
| 5    | R/W  | 读/写选择      |
| 6    | E    | 使能信号       |
| 7    | D0   | 数据位0        |
| 8    | D1   | 数据位1        |
| 9    | D2   | 数据位2        |
| 10   | D3   | 数据位3        |
| 11   | D4   | 数据位4        |
| 12   | D5   | 数据位5        |
| 13   | D6   | 数据位6        |
| 14   | D7   | 数据位7        |
| 15   | A    | 背光阳极（+）  |
| 16   | K    | 背光阴极（-）  |

## 6. 接口信号

### 6.1 RS（寄存器选择）

| RS | 功能             |
|----|------------------|
| 0  | 指令寄存器       |
| 1  | 数据寄存器       |

### 6.2 R/W

| R/W | 操作     |
|-----|----------|
| 0   | 写入LCD  |
| 1   | 从LCD读取 |

### 6.3 使能（E）

* 数据在 **E的下降沿** 被锁存
* 每次传输时E必须从高电平变为低电平

---

## 7. 数据总线操作

### 7.1 8位模式

* 使用D0-D7
* 操作更快

### 7.2 4位模式

* 仅使用D4-D7
* 数据以两个半字节（高位优先）传输
* 节省I/O引脚

---

## 8. 内部内存映射

### 8.1 DDRAM（显示数据RAM）

| 地址 | 显示位置 |
|------|----------|
| 0x00-0x0F | 第1行，第1-16列 |
| 0x40-0x4F | 第2行，第1-16列 |

### 8.2 CGRAM（字符生成RAM）

* 支持最多 **8个自定义字符**
* 每个字符占用8字节

---

## 9. 指令集（概要）

| 指令 | 描述            |
|------|-----------------|
| 0x01 | 清除显示        |
| 0x02 | 返回初始位置    |
| 0x04-0x07 | 入口模式设置 |
| 0x08-0x0F | 显示开关控制 |
| 0x10-0x1F | 光标/显示移位 |
| 0x20-0x3F | 功能设置       |
| 0x40-0x7F | 设置CGRAM地址  |
| 0x80-0xFF | 设置DDRAM地址  |

---

## 10. 初始化序列（4位模式）

```text
VDD上升后等待>15ms
功能设置：0x33
功能设置：0x32
功能设置：0x28（4位，2行）
显示开关：0x0C
入口模式：0x06
清除显示：0x01
```

## 12. 典型系统集成（6502示例）

```text
RS   VIA输出
E    VIA输出
D4-D7  VIA端口B
R/W  接地（仅写）
```

---

## 13. 对比度和背光控制

* 对比度通过VO引脚上的电位器调节
* 背光可能需要串联电阻
* 支持通过外部控制实现PWM调光
