# AT28C256 256K（32K×8）并行EEPROM规格书

## 1. 概述

**AT28C256** 是由Atmel（现为Microchip）制造的非易失性、电可擦除和可编程只读存储器（EEPROM）。它提供 **256 Kbit** 的存储空间，组织为 **32,768×8位**，常用于基于 **6502**、**Z80** 和类似CPU的8位微处理器系统中。

该设备支持字节级写入操作、快速读取访问以及软件控制的数据保护。

---

## 2. 一般特性

| 特性         | 描述                         |
|--------------|------------------------------|
| 存储容量     | 256 Kbit（32 KB）             |
| 组织方式     | 32,768×8位                    |
| 数据总线     | 8位                          |
| 地址总线     | 15位（A0-A14）                 |
| 技术类型     | EEPROM                       |
| 耐久性       | ≥100,000次写入周期              |
| 数据保持时间 | ≥10年                        |
| 访问时间     | 150-250 ns（视变种而定）        |
| 封装类型     | DIP-28、PLCC-32、TSOP           |

---

## 3. 引脚配置（逻辑）

### 3.1 地址引脚（A0-A14）

* 选择32,768个存储单元中的一个

### 3.2 数据引脚（I/O0-I/O7）

* 双向三态数据总线
* 读取周期中输出有效数据

### 3.3 控制引脚

| 引脚 | 描述                         |
|------|------------------------------|
| CE   | 芯片使能（低电平有效）          |
| OE   | 输出使能（低电平有效）          |
| WE   | 写入使能（低电平有效）           |
| VCC  | +5 V电源                      |
| GND  | 接地                         |

---

## 4. 存储器组织

* 线性地址空间从 `$0000` 到 `$7FFF`
* 每个地址对应一个8位字节

```text
地址范围：0000h - 7FFFh
数据宽度：8位
```

---

## 5. 读取操作

### 5.1 读取周期条件

| 信号 | 状态 |
|------|-----|
| CE   | 低电平 |
| OE   | 低电平 |
| WE   | 高电平 |

* 数据在访问时间后出现在I/O引脚上
* 当CE和OE被激活时，输出保持有效
* 当CE或OE为高电平时，输出为高阻抗状态

---

## 6. 写入操作

### 6.1 字节写入周期

| 信号 | 状态     |
|------|---------|
| CE   | 低电平   |
| OE   | 高电平   |
| WE   | 低电平脉冲 |

* 在WE低电平脉冲期间，地址和数据必须稳定
* 内部写入周期时间约为10 ms（最大值）
* 设备自动处理写入前的擦除操作

---

## 7. 软件数据保护（SDP）

AT28C256包含可选的**软件数据保护**功能，以防止意外写入。

### 7.1 SDP使能序列

```text
向地址$5555写入$AA
向地址$2AAA写入$55
向地址$5555写入$A0
```

### 7.2 SDP禁用序列

```text
向地址$5555写入$AA
向地址$2AAA写入$55
向地址$5555写入$80
向地址$5555写入$AA
向地址$2AAA写入$55
向地址$5555写入$20
```

---

## 8. 写入周期定时说明

* 写入操作由内部定时控制，无需外部轮询
* 写入周期期间，读取操作返回未定义数据
* 设备在忙碌时忽略额外的写入尝试

---

## 9. 数据轮询（可选）

* 在写入过程中可监控I/O7
* 当I/O7与写入的数据匹配时，写入完成

---

## 10. 复位与电源行为

* 无专用复位引脚
* 上电和断电期间禁止写入操作
* 输出默认为高阻抗状态，直到CE和OE被激活

---

## 11. 典型系统集成（以6502为例）

```text
地址范围：$8000 - $FFFF
A15引脚用作芯片选择
OE  读写控制？

```
