# 常用面包板电子元件

**基于面包板的电子实用参考手册**，涵盖常见元件、公式、表格、引脚排列及学习参考资料。适合从初学者到中级硬件构建者。

---

## 1. 电阻器

### 描述

电阻器用于限制电流、分压和设定偏置点。

### 常见类型

| 类型         | 备注       | 典型用途         |
| ------------ | ---------- | ---------------- |
| 碳膜电阻     | 价格低廉，±5% | 通用用途         |
| 金属膜电阻   | 稳定，±1%   | 精密电路         |
| 绕线电阻     | 高功率     | 功率耗散         |
| 电位器       | 可调       | 音量调节、调谐   |

### 标准值（E12系列）

`10, 12, 15, 18, 22, 27, 33, 39, 47, 56, 68, 82 × 10^n`

### 色环编码

| 色环 | 含义         |
| ---- | ------------ |
| 1    | 第一位数字   |
| 2    | 第二位数字   |
| 3    | 倍率         |
| 4    | 容差         |

### 功率等级

| 等级 | 典型尺寸         |
| ---- | ---------------- |
| 1/8 W | 小型             |
| 1/4 W | 面包板标准       |
| 1/2 W | 大型             |

### 公式

* **欧姆定律：** `V = I × R`
* **功率：** `P = V × I = I² × R = V² / R`

### 参考资料

* [https://en.wikipedia.org/wiki/Resistor](https://en.wikipedia.org/wiki/Resistor)

## 2. 电容器

### 描述

电容器用于存储电能，并阻止直流电通过而允许交流电通过。

### 常见类型

| 类型         | 是否有极性 | 典型用途         |
| ------------ | ---------- | ---------------- |
| 陶瓷电容     | 否         | 去耦、旁路       |
| 电解电容     | 是         | 大容量滤波       |
| 钽电容       | 是         | 小型滤波         |
| 薄膜电容     | 否         | 信号耦合         |

### 常见容值

| 范围 | 示例             |
| ---- | ---------------- |
| pF   | 10pF, 100pF      |
| nF   | 100nF（0.1µF）   |
| µF   | 1µF, 10µF, 100µF |

### 电压等级规则

> 电容的电压等级 ≥ **电路电压的2倍**

### 公式

* **电容抗：** `Xc = 1 / (2πfC)`
* **储存能量：** `E = ½ C V²`

### 参考资料

* [https://en.wikipedia.org/wiki/Capacitor](https://en.wikipedia.org/wiki/Capacitor)

## 3. 电感器

### 描述

电感器通过磁场存储能量，并抵抗电流变化。

### 常见类型

| 类型         | 用途             |
| ------------ | ---------------- |
| 空芯电感器   | 射频电路         |
| 铁氧体芯电感器 | 电源滤波         |
| 扼流圈       | 抑制噪声         |

### 公式

* **电感抗：** `XL = 2πfL`

### 参考资料

* [https://en.wikipedia.org/wiki/Inductor](https://en.wikipedia.org/wiki/Inductor)

## 4. 二极管

### 描述

二极管仅允许电流单向流动。

### 常见类型

| 类型       | 典型型号 | 用途             |
| ---------- | -------- | ---------------- |
| 整流二极管 | 1N4007   | 电源应用         |
| 信号二极管 | 1N4148   | 逻辑、快速开关   |
| 齐纳二极管 | 1N4733A  | 电压调节         |
| 肖特基二极管 | 1N5819 | 低压降应用       |

### 关键参数

| 参数         | 典型值                     |
| ------------ | --------------------------- |
| 正向电压     | 0.7V（硅），0.3V（肖特基）  |
| 反向电压     | 由器件决定                 |

### 参考资料

* [https://en.wikipedia.org/wiki/Diode](https://en.wikipedia.org/wiki/Diode)

## 5. LED（发光二极管）

### 描述

LED在正向偏置时会发光。

### 典型正向电压

| 颜色 | Vf         |
| ---- | ---------- |
| 红色 | 1.8-2.2V   |
| 绿色 | 2.0-3.0V   |
| 蓝色 | 3.0-3.3V   |

### 限流电阻

`R = (V_supply - V_LED) / I_LED`

### 参考资料

* [https://en.wikipedia.org/wiki/Light-emitting_diode](https://en.wikipedia.org/wiki/Light-emitting_diode)

## 6. 晶体管

### BJT（双极型晶体管）

| 类型 | 示例 | 用途                   |
| ---- | ---- | ----------------------- |
| NPN  | 2N3904 | 开关、放大             |
| PNP  | 2N3906 | 高边开关               |

* `Ic ≈ β × Ib`

## 7. 集成电路（DIP）

### 常见逻辑系列

| 系列       | 电压 | 备注             |

## 9. 面包板

### 描述

无焊接原型板，具有内部总线连接。

### 内部布线

* 行：水平连接（5个孔位）
* 轨道：垂直连接（电源）

### 局限性

* 不适合高频或大电流电路

### 参考资料

* [https://en.wikipedia.org/wiki/Breadboard](https://en.wikipedia.org/wiki/Breadboard)
