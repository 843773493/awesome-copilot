# T48通用EEPROM/闪存编程器规格说明书

## 1. 概述

**T48通用编程器**（也称为**TL866-3G / TL866II Plus后续型号**）是一款通过USB连接的设备，用于**读取、编程和验证**多种**EEPROM、闪存、EPROM、GAL和微控制器设备**。它常用于**复古计算机**、**固件开发**和**电子维修**工作流程中。

T48是TL866系列的现代替代品，提供了更广泛的设备支持和改进的软件兼容性。

---

## 2. 一般特性

| 特性             | 描述                          |
| ------------------ | ----------------------------- |
| 编程器类型         | 通用设备编程器                |
| 接口               | USB 2.0                      |
| 插座               | ZIF-48                       |
| 支持的设备类型     | EEPROM、Flash、EPROM、GAL、MCU |
| 编程电压           | 内部生成                      |
| 主机操作系统       | Windows、Linux（开源工具）     |
| 验证方式           | 写入后读取、校验和            |

---

## 3. 外观描述

### 3.1 ZIF插座

* **48引脚零插入力（ZIF）插座**
* 支持直接插入DIP封装
* 通过适配器支持PLCC、SOP、TSOP封装

### 3.2 指示灯与控制

| 项目             | 描述                         |
| ------------------ | ---------------------------- |
| 状态指示灯         | 电源/活动指示                |
| ZIF锁扣           | 锁定IC至插座                 |
| USB连接器         | 主机连接和供电               |

---

## 4. 支持的设备类型

### 4.1 存储设备

* 27xxx EPROM
* 28xxx EEPROM（例如：AT28C256）
* 29xxx Flash存储器
* 串行EEPROM（需适配器支持）

### 4.2 可编程逻辑

* GAL16V8、GAL22V10（读取/写入/验证）

### 4.3 微控制器（有限支持）

* 部分PIC、AVR和8051系列设备
* 编程支持取决于电压和引脚配置

---

## 5. 电气性能

| 参数              | 描述                          |
| ------------------ | ----------------------------- |
| VCC电压范围       | 1.8 V - 6.5 V（设备相关）       |
| VPP电压           | 内部生成，设备特定              |
| 输入/输出保护     | 过电流和短路保护               |

---

## 6. 编程操作

### 6.1 常规操作

* 设备识别
* 空白检查
* 读取
* 编程（写入）
* 验证
* 擦除（适用于Flash/EPROM）

### 6.2 验证模式

* 按字节比较
* 校验和/ CRC校验

---

## 7. 软件支持

### 7.1 官方软件

* 基于Windows的图形用户界面
* 带有引脚映射的设备数据库
* 自动电压和时序控制

### 7.2 开源工具

| 工具       | 备注                          |
| ----------- | ----------------------------- |
| `minipro`  | Linux / macOS / Windows命令行工具 |
| libusb     | USB通信后端                   |

示例：

```bash
minipro -p AT28C256 -w rom.bin
```

---

## 8. 设备插入与引脚映射

* 设备方向由**引脚1标记**指示
* 许多设备在ZIF插座中使用**左下对齐**
* 软件显示正确的插入示意图

---

## 9. 典型工作流程（以EEPROM为例）

1. 选择设备类型（例如：AT28C256）
2. 将芯片插入ZIF插座
3. 进行空白检查
4. 加载二进制镜像
5. 编程设备
6. 验证写入内容

---

## 10. 电源与安全注意事项

* 编程器完全通过USB供电
* 编程过程中不得插入或移除IC
* 非DIP封装设备需使用适配器

---

## 11. 局限性

* 并非所有微控制器均支持
* 高压EPROM可能需要特定适配器
* 不适用于电路内编程（ISP）

---

## 12. 常见使用场景

* 为6502单板计算机编程EEPROM
* 为复古系统烧录ROM镜像
* 读取和备份旧式EPROM
* GAL逻辑开发

---

| 操作系统支持     | Windows      | Windows / Linux |
