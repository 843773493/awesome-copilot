# 基本电子元件

本文件涵盖四种基本电子元件：电阻器、电容器、按钮和开关，包括常见的数值、公式和类型。

---

## 1. 电阻器 (R)

电阻器限制电流的流动，并将能量以热的形式耗散。

### 常见公式

* **欧姆定律:** $V = I \times R$ (电压 = 电流 $\times$ 电阻)
* **串联电阻:** $R_{total} = R_1 + R_2 + ... + R_n$
* **并联电阻:** $\frac{1}{R_{total}} = \frac{1}{R_1} + \frac{1}{R_2} + ... + \frac{1}{R_n}$

### 常见数值 (E12/E24系列 - 5%公差)

数值通过10的幂次进行倍增（例如：10$\Omega$，100$\Omega$，1k$\Omega$，10k$\Omega$，100k$\Omega$，1M$\Omega$）：
**1.0, 1.1, 1.2, 1.3, 1.5, 1.6, 1.8, 2.0, 2.2, 2.4, 2.7, 3.0, 3.3, 3.6, 3.9, 4.3, 4.7, 5.1, 5.6, 6.2, 6.8, 7.5, 8.2, 9.1**

## 2. 电容器 (C)

电容器在电场中存储电荷。

### 常见公式

* **电容:** $C = \frac{Q}{V}$ (电荷 / 电压)
* **电流:** $i = C \frac{dV}{dt}$
* **并联电容:** $C_{total} = C_1 + C_2 + ... + C_n$
* **串联电容:** $\frac{1}{C_{total}} = \frac{1}{C_1} + \frac{1}{C_2} + ... + \frac{1}{C_n}$
* **储存能量:** $W = \frac{1}{2} C V^2$

### 常见数值

* **陶瓷电容（皮法拉/纳法拉）:** 10pF, 22pF, 100pF, 1nF, 10nF, 100nF（通常标记为"104"）
* **电解电容（微法拉）:** 1µF, 10µF, 22µF, 47µF, 100µF, 220µF, 470µF, 1000µF

---

## 3. 按钮（推钮）

一种瞬时开关，仅在按下时闭合电路。

### 常见类型

* **常开（NO）:** 电路处于断开状态（关闭）直至被按下。
* **常闭（NC）:** 电路处于闭合状态（开启）直至被按下。
* **触点开关:** 用于PCB安装的小型标准按钮。

---

## 4. 开关（SW）

一种机电装置，用于断开或连接电路，直到被翻转才会保持位置。

### 常见类型

* **SPST:** 单刀单掷（通断）
* **SPDT:** 单刀双掷（在两个路径之间切换）
* **DPST/DPDT:** 双刀变体（同时控制两个独立电路）
* **DIP开关:** 用于电路板的小型开关阵列。

## 快速参考总结表

| 元件 | 符号（参考） | 关键公式 | 常见用途 |
| :--- | :--- | :--- | :--- |
| **电阻器** | 锯齿形/方框 | $V=IR$ | 电流限制、电压分压 |
| **电容器** | 两条平行线 | $i=C \frac{dV}{dt}$ | 滤波、定时、耦合 |
| **按钮** | 瞬时 | 无 | 用户输入（瞬时） |
| **开关** | 拨动/杠杆 | 无 | 电源控制、信号路由 |
