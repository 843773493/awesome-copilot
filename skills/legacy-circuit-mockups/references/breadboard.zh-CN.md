# 无焊接面包板

一款实用的Markdown规范和参考指南，用于**常见的无焊接面包板**，适用于电子原型设计、教学和业余爱好者开发。

---

## 1. 概述

**无焊接面包板**是一种可重复使用的原型设计平台，允许电子元件在无需焊接的情况下相互连接。连接通过内部弹簧夹实现。

### 典型用途

* 快速电路原型设计
* 教学实验室
* 逻辑和微控制器实验
* 低功耗模拟和数字电路

### 不适合的场景

* 高电流（>1A）
* 高电压（>36V）
* 射频/高频设计
* 易受振动影响或固定安装

---

## 2. 物理结构

### 材料

* ABS或聚苯乙烯主体
* 磷铜或镀镍弹簧触点
* 粘贴背板（可选）

### 标准孔距

* **2.54 mm（0.1英寸）** - 与DIP集成电路和标准引脚座兼容

### 接触特性

| 参数          | 典型值 |
| ------------- | ------ |
| 接触电阻      | 10-50 毫欧 |
| 插入次数       | ~5,000 次 |
| 导线规格       | 20-28 AWG |

---

## 3. 内部电气连接

### 接线端子（主区域）

* 每行有**5个相互连接的孔**
* 水平连接
* 中心沟槽将左右两半隔离

```
A B C D E | F G H I J
──────────┼──────────
连接      | 连接
```

### 电源轨道

* 每侧有垂直总线
* 通常**在中间断开**（并非总是连续）
* 通常标记为**红色（+）**和**蓝色（-）**

```
+  +  +  +  +   （可能断开）
-  -  -  -  -
```

> ? 请始终使用万用表验证连续性

## 5. 元件兼容性

### 兼容元件

* DIP集成电路（300 mil，600 mil）
* 轴向电阻器和二极管
* 发光二极管（LED）
* 按键开关
* 跳线
* 引脚座

### 不兼容元件

| 元件             | 问题                      |
| ---------------- | -------------------------- |
| TO-220封装        | 宽度太大/触点受应力         |
| SMD元件          | 需要适配器                 |
| 大电解电容        | 机械应力                   |
| 高功率电阻器      | 发热                      |

## 6. 电气特性

### 典型限制

| 参数            | 建议最大值     |
| --------------- | ------------- |
| 电压             | 30-36 V       |
| 每个触点电流     | 500-1000 mA   |
| 频率             | <1 MHz（实用）|

### 寄生参数（近似值）

| 类型        | 值               |
| ----------- | ---------------- |
| 电容        | 每个节点1-5 pF   |
| 电感        | 每根跳线10-20 nH |

---

## 7. 最佳实践

### 电源分配

* 将**接地和Vcc**连接到两侧
* 如有需要，可桥接断开的电源轨道
* 使用**0.1µF陶瓷电容**对集成电路进行去耦

### 线缆布线

* 保持线缆短而整洁
* 使用颜色编码：

  * 红色：Vcc
  * 黑色/蓝色：GND
  * 黄色/白色：信号

### 集成电路布局

* 将DIP集成电路**跨过中心沟槽**放置
* 避免强行插入引脚

---

## 8. 常见错误

| 错误                       | 结果             |
| -------------------------- | -------------- |
| 假设电源轨道是连续的        | 电源损耗         |
| 使用过长的跳线             | 噪声、不稳定     |
| 未使用去耦电容             | 不稳定行为       |
| 超出电流限制               | 触点熔化         |

---

## 9. 测试与调试

### 连续性检查

* 使用万用表验证电源轨道和行连接

### 信号完整性提示

* 避免使用面包板进行：

  * 高速时钟
  * ADC精度电路
