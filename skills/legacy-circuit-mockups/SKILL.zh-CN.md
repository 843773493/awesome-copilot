# 遗留电路草图

一种用于创建复古计算和电子项目面包板电路草图和视觉示意图的技能。该技能利用HTML5 Canvas绘图机制，渲染包含复古元件（如6502微处理器、555定时器IC、EEPROM和7400系列逻辑门）的交互式电路布局。

## 何时使用此技能

- 用户要求“创建面包板布局”或“制作电路草图”
- 用户希望在面包板上可视化元件放置
- 用户需要构建6502计算机的视觉参考
- 用户要求“绘制电路”或“电子元件示意图”
- 用户希望创建教育用电子视觉图
- 用户提到Ben Eater教程或复古计算项目
- 用户要求制作555定时器电路或LED项目草图
- 用户需要可视化元件间的导线连接

## 先决条件

- 理解捆绑参考文件中的元件引脚配置
- 熟悉面包板布局规范（行、列、电源轨）

| 28C256 | 28引脚DIP | 32KB并行EEPROM |
| W65C22 | 40引脚DIP | 可变接口适配器（VIA） |
| 62256 | 28引脚DIP | 32KB静态RAM |

## 分步工作流程

### 创建基本LED电路草图

1. 定义面包板尺寸和网格
2. 连接电源轨（+5V和GND）
3. 添加LED元件并设置阳极/阴极方向
4. 放置限流电阻器
5. 在元件间绘制导线连接
6. 添加标签和注释

### 创建555定时器电路

1. 将NE555 IC放置在面包板中央通道两侧
2. 将引脚1（GND）连接到地轨
3. 将引脚8（Vcc）连接到电源轨
4. 添加定时电阻器和电容器
5. 连接触发和阈值引脚
6. 将引脚3（输出）连接到LED或其他负载

### 创建6502微处理器布局

1. 将W65C02S微处理器居中放置在面包板上
2. 添加28C256 EEPROM用于程序存储
3. 放置W65C22 VIA用于I/O
4. 添加7400系列逻辑用于地址解码
5. 连接地址总线（A0-A15）
6. 连接数据总线（D0-D7）
7. 连接控制信号（R/W, PHI2, RESB）
8. 添加复位按钮和时钟晶体

## 元件引脚速查表

### 555定时器（8引脚DIP）

| 引脚 | 名称 | 功能 |
|:----:|:-----|:-----|
| 1 | GND | 接地（0V） |
| 2 | TRIG | 触发（低于1/3 Vcc启动定时） |
| 3 | OUT | 输出（可提供/吸收200mA电流） |
| 4 | RESET | 低电平复位 |
| 5 | CTRL | 控制电压（需通过10nF电容旁路） |
| 6 | THR | 阈值（高于2/3 Vcc重置） |
| 7 | DIS | 放电（开漏输出） |
| 8 | Vcc | 电源（+4.5V至+16V） |

### W65C02S（40引脚DIP）- 关键引脚

| 引脚 | 名称 | 功能 |
|:----:|:-----|:-----|
| 8 | VDD | 电源 |
| 21 | VSS | 接地 |
| 37 | PHI2 | 系统时钟输入 |
| 40 | RESB | 低电平复位 |
| 34 | RWB | 读/写信号 |
| 9-25 | A0-A15 | 地址总线 |
| 26-33 | D0-D7 | 数据总线 |

### 28C256 EEPROM（28引脚DIP）- 关键引脚

| 引脚 | 名称 | 功能 |
|:----:|:-----|:-----|
| 14 | GND | 接地 |
| 28 | VCC | 电源 |

| 27 | WE | 写使能（低电平有效） |
| 1-10, 21-26 | A0-A14 | 地址输入 |
| 11-19 | I/O0-I/O7 | 数据总线 |

## 公式参考

### 电阻器计算

- **欧姆定律：** V = I × R
- **LED电流：** R = (Vcc - Vled) / Iled
- **功率：** P = V × I = I² × R

### 555定时器公式

**无稳态模式：**

- 频率：f = 1.44 / ((R1 + 2×R2) × C)
- 高电平时间：t₁ = 0.693 × (R1 + R2) × C
- 低电平时间：t₂ = 0.693 × R2 × C
- 占空比：D = (R1 + R2) / (R1 + 2×R2) × 100%

**单稳态模式：**

- 脉冲宽度：T = 1.1 × R × C

### 电容器计算

- 电容抗：Xc = 1 / (2πfC)
- 存储能量：E = ½ × C × V²

## 构建示例

### 构建1 — 单个LED

**元件：** 红色LED、220Ω电阻器、跳线、电源

**步骤：**

1. 将黑色跳线从电源GND插入到A5行
2. 将红色跳线从电源+5V插入到J5行
3. 将LED的阴极（短腿）对齐到GND行放置
4. 在电源和LED阳极之间放置220Ω电阻器

### 构建2 — 555无稳态闪烁器

**元件：** NE555、LED、电阻器（10kΩ、100kΩ）、电容器（10µF）

**步骤：**

1. 将555 IC放置在面包板中央通道上
2. 将引脚1连接到GND，引脚8连接到+5V
3. 将引脚4连接到引脚8（禁用复位）
4. 在引脚7和+5V之间连接10kΩ电阻器
5. 在引脚6和7之间连接100kΩ电阻器
6. 在引脚6和GND之间连接10µF电容器
7. 将引脚3（输出）连接到LED电路

## 常见问题排查

| 问题 | 解决方案 |
|------|----------|
| LED不亮 | 检查极性（阳极接+，阴极接-） |
| 电路无电源 | 验证电源轨连接 |
| IC不工作 | 检查VCC和GND引脚连接 |
| 555不振荡 | 验证阈值/触发电容器连接 |
| 微处理器卡死 | 检查复位脉冲后RESB是否为高电平 |
