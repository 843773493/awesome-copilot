

# 视觉检查清单

本文档是进行网页设计视觉检查时需要验证的项目综合清单。

---

## 1. 布局验证

### 结构完整性

- [ ] 页眉正确固定/定位在屏幕顶部
- [ ] 页脚定位在屏幕底部或内容末尾
- [ ] 主内容区域居中对齐且宽度合适
- [ ] 侧边栏（如有）位置正确
- [ ] 导航栏显示在预期位置

### 溢出

- [ ] 没有意外显示水平滚动条
- [ ] 内容不会溢出父元素
- [ ] 图片适合父容器
- [ ] 表格宽度不超过容器宽度
- [ ] 代码块正确换行或滚动

### 对齐

- [ ] 网格项目均匀分布
- [ ] 弹性项目对齐正确
- [ ] 文字对齐（左/中/右）一致
- [ ] 图标和文字垂直对齐
- [ ] 表单标签和输入框位置正确

---

## 2. 字体验证

### 可读性

- [ ] 正文文字字体大小足够（建议最小16px）
- [ ] 行高合适（建议1.5-1.8）
- [ ] 每行字符数合适（建议40-80个字符）
- [ ] 段落之间间距足够
- [ ] 标题大小层级清晰

### 文字处理

- [ ] 长单词正确换行
- [ ] URLs和代码正确处理
- [ ] 没有文字裁剪现象
- [ ] 省略号（...）正确显示
- [ ] 语言特定的换行规则正常工作

### 字体

- [ ] 网页字体正确加载
- [ ] 备用字体合适
- [ ] 字体粗细符合预期
- [ ] 特殊字符和表情符号正确显示

---

## 3. 颜色与对比度验证

### 可访问性（WCAG标准）

- [ ] 正文文字：对比度比4.5:1或更高（AA）
- [ ] 大文字（18px+加粗或24px+）：对比度比3:1或更高
- [ ] 交互元素边框：对比度比3:1或更高
- [ ] 聚焦指示器：与背景有足够对比度

### 颜色一致性

- [ ] 品牌颜色统一
- [ ] 链接颜色一致
- [ ] 错误状态红色统一
- [ ] 成功状态绿色统一
- [ ] 悬停/激活状态颜色合适

### 色觉多样性

- [ ] 信息通过形状和文字传达，而非仅通过颜色
- [ ] 图表和图示考虑色觉多样性
- [ ] 错误信息不依赖颜色单独传达

---

## 4. 响应式验证

### 移动端（约640px）

- [ ] 内容适合屏幕宽度
- [ ] 触摸目标尺寸至少为44x44px
- [ ] 文字可读尺寸
- [ ] 没有水平滚动
- [ ] 导航适合移动端（汉堡菜单等）
- [ ] 表单输入易于使用

### 平板（641px~1024px）

- [ ] 布局优化为平板设备
- [ ] 两列布局显示合适
- [ ] 图片尺寸合适
- [ ] 侧边栏显示/隐藏合适

### 桌面端（1025px~）

- [ ] 设置最大宽度，不会在超大屏幕上断裂
- [ ] 间距足够
- [ ] 多列布局功能正常
- [ ] 悬停状态已实现

### 突变点过渡

- [ ] 屏幕尺寸变化时布局平滑过渡
- [ ] 中间尺寸下布局不会断裂
- [ ] 没有内容消失或重复

---

## 5. 交互元素验证

### 按钮

- [ ] 默认状态清晰
- [ ] 桌面端有悬停状态
- [ ] 聚焦状态视觉清晰
- [ ] 有激活（按下）状态
- [ ] 禁用状态可区分
- [ ] 有加载状态（如适用）

### 链接

- [ ] 链接可视觉识别
- [ ] 已访问链接可区分（如需要）
- [ ] 有悬停状态
- [ ] 聚焦状态清晰

### 表单元素

- [ ] 输入框边界清晰
- [ ] 占位符文字对比度合适
- [ ] 聚焦时有视觉反馈
- [ ] 错误状态显示
- [ ] 必填字段标识
- [ ] 下拉菜单功能正常

---

## 6. 图片与媒体验证

### 图片

- [ ] 图片显示在合适尺寸
- [ ] 保持宽高比
- [ ] 支持高分辨率显示（@2x）
- [ ] 图片加载失败时显示替代内容
- [ ] 懒加载行为正常

### 视频与嵌入内容

- [ ] 视频适合容器
- [ ] 保持宽高比
- [ ] 嵌入内容响应式
- [ ] iframes不溢出

---

## 7. 可访问性验证

### 键盘导航

- [ ] 所有交互元素可通过Tab键访问
- [ ] 聚焦顺序逻辑合理
- [ ] 聚焦陷阱适当（模态框等）
- [ ] 存在“跳过到内容”链接

### 屏幕阅读器支持

- [ ] 图片有alt文本
- [ ] 表单有标签
- [ ] ARIA标签适当设置
- [ ] 标题层级正确（h1→h2→h3...）

### 动画

- [ ] 动画不过于频繁
- [ ] 支持prefers-reduced-motion（如可能）

---

## 优先级矩阵

| 优先级 | 类别 | 示例 |
|--------|------|------|
| P0（关键） | 功能破坏 | 元素完全重叠，内容消失 |
| P1（高） | 严重用户体验问题 | 不可读文字，无法操作的按钮 |
| P2（中） | 中等问题 | 对齐问题，间距不一致 |
| P3（低） | 小问题 | 轻微位置差异，小颜色变化 |

## 验证工具

### 浏览器开发者工具

- 元素面板：DOM和样式检查
- Lighthouse：性能和可访问性审核
- 设备工具栏：响应式测试

### 可访问性工具

- axe 开发工具
- WAVE
- 颜色对比度分析器

### 自动化工具

- Playwright（截图对比）
- Percy / Chromatic（视觉回归测试）
