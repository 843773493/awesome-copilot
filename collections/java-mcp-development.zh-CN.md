## 集合使用方式

### Java MCP 专家

推荐使用

此聊天模式为构建Java MCP服务器提供专家指导。

此模式适用于：
- 创建新的Java MCP服务器项目
- 使用Project Reactor实现反应式处理器
- 配置stdio或HTTP传输
- 调试反应式流和错误处理
- 学习使用官方SDK的Java MCP最佳实践
- 与Spring Boot应用程序集成

为了获得最佳效果，请考虑：
- 使用指令文件为Java MCP开发设置上下文
- 使用提示生成初始项目结构
- 切换到专家聊天模式以获取详细实现帮助
- 指定是否需要Maven或Gradle
- 提供所需工具或功能的详细信息
- 说明是否需要Spring Boot集成
