## 集合使用方式

### Swift MCP 专家

推荐

此聊天模式为构建 MCP 服务器提供专家指导。

此模式适合以下场景：
- 使用 Swift 创建新的 MCP 服务器项目
- 实现异步/等待模式和基于 actor 的并发
- 设置 stdio、HTTP 或网络传输
- 调试 Swift 并发和 ServiceLifecycle 集成
- 学习使用官方 SDK 的 Swift MCP 最佳实践
- 优化 iOS/macOS 平台服务器性能

为获得最佳效果，建议：
- 使用指令文件为 Swift MCP 开发设置上下文
- 使用提示生成初始项目结构
- 切换到专家聊天模式以获取详细实现帮助
- 指定是否需要 stdio、HTTP 或网络传输
- 提供所需工具或功能的详细信息
- 提及是否需要资源、提示或特殊功能
