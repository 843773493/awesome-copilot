# 开源资助

为开源项目办公室（OSPOs）提供的工具和资源，用于通过 GitHub Sponsors、Open Collective 和其他资助平台识别、评估和管理开源依赖项的赞助。

**标签:** ospo, 赞助, 开源, 资助, github-sponsors

## 此集合中的项目

| 标题 | 类型 | 描述 |
|------|------|------|
| [Sponsor Finder](../skills/sponsor-finder/SKILL.md) | 技能 | 查找某个 GitHub 仓库的依赖项中哪些可以通过 GitHub Sponsors 赞助。使用 deps.dev API 进行 npm、PyPI、Cargo、Go、RubyGems、Maven 和 NuGet 等包管理器的依赖解析。检查 npm 赞助元数据、FUNDING.yml 文件和网络搜索。验证所有链接。显示直接和传递依赖项，并结合 OSSF Scorecard 健康数据。通过提供 GitHub 仓库的所有者/仓库名称（例如 "查找 expressjs/express 仓库中可赞助的依赖项"）来调用。[查看用法](#sponsor-finder) |

## 集合使用方式

### 赞助查找器

查找某个 GitHub 仓库的依赖项中哪些可以通过 GitHub Sponsors 赞助。
通过提供 GitHub 仓库的所有者/仓库名称（例如 "查找 expressjs/express 仓库中可赞助的依赖项"）来调用。
